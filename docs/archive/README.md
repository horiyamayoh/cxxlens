# Documentation archive

この directory の本文はすべて `cxxlens_authority: non-normative` です。過去の要求、受入証跡、設計判断を
監査・移行するためだけに保持し、新規 API/relation/provider の契約根拠には使用しません。

- `legacy-v1/design/` — use-case 固定 API、package contract、旧統合設計
- `legacy-v1/development/` — central fact/store/search/workspace の旧実装契約
- `legacy-v1/agent/` — 旧 124 API task packet/readiness/dispatch
- `pre-cxxlens/` — cxxlens 統合以前の構想

各 archive 文書は replacement と removal issue を front matter に持ちます。移行判断は
[asset migration ledger](../../schemas/cxxlens_asset_migration_ledger.json) を参照してください。
